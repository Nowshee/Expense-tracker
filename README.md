# Smart Expense Tracker (Python)

A simple command-line based Expense Tracker built using Python.
This project allows users to record daily expenses, view all records,
and calculate the total amount spent.

## Features
- Add new expenses (Category, Amount, Date)
- View all saved expenses
- Show total expense
- Data stored permanently in a text file (`expense.txt`)
- Menu-driven interface using loops and functions

## How to Run
1. Open `Smart_Expense_Tracker.ipynb` in Jupyter Notebook.
2. Run all cells.
3. Use the menu:
   - Press 1 to Add Expense  
   - Press 2 to View All Expenses  
   - Press 3 to Show Total  
   - Press 4 to Exit
